# Supervisor Monitor
[![Build Status](https://travis-ci.org/naginn)](https://travis-ci.org/supervisor-monitor)

## Features

- Easy admin panel for supervisor
- Drag and drop

## Installation

```sh
pip install supervisor-monitor
```

#### Environments

```sh
export SUPERVISOR_CONF=/your/path/to
```


#### Run

For production release:

```sh
python run.py
```
